# Getting Started with Rock-Scissors-Paper App

1. Clone this repository. <br>
```
git clone https://github.com/takoxz/rock-scissors-paper
```

2. Change directory 
```
cd rock-scissors-paper
```

3. Install packages
```
npm install package.json
```

4. Run the App
```
npm start
```

5. Visit the site
```
Go to http://localhost:3000/
```
